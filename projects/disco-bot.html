About

     is a chat bot made for Discord, specializing in information and statistics on Riot Games' League of Legends. Users can play music from Youtube and Soundcloud, search for LoL player statistics and information, search for LoL in-game data, and search for LoL game statistics. Disco Bot uses various APIs and libraries to grab LoL data and upload it to the Discord text channels.

Usage

Basic Usage and Commands

    To use , you must first invite the bot to your Discord server. Run via command line, and then will be up and running. Users can enter commands, pre-pended with "\." The commands are as follows:

  • play, queue: Queues up a given Youtube or Soundcloud url or search query.
  • volume, vol: Changes the volume of the music player.
  • skip, next: Starts a vote to skip the current song. Majority is needed to skip, unless the song requester requested to skip.
  • pause: Pauses the current song.
  • stop, leave: Stops the music player, and removes all songs from the current playlist.
  • list, songs: Lists all of the songs currently in queue.
  • current, song: Displays info on the current song.
  • summoner: Searches for a given LoL player name, and displays information on the player.
  • mastery: Searches for a given LoL player name, and displays his/her mastery pages. Note: Deprecated.
  • champion: Searches for a given LoL champion, and displays champion information.
  • item: Searches for a given LoL item, and displays item information.
  • stats: Searches for a given LoL champion, and displays champion play statistics.
  • top: Gets the top X champions in a given role.
  • add: Adds two given integers.
  • multiply: Multiplies two given integers.
  • roll: Rolls the given dice, in die notation.
  • mean: Calculates the mean average of the given dice, in die notation.
  • choose: Randomly chooses an item among the given items.

Optional Arguments

    Many commands have optional arguments as well. The summoner command can accept an optional region argument, informing which region to search the player in. By default, it searches under the NA region. The champion command accepts a few optional arguments: -i, -st, -s, -t, -l, -sk. These arguments tell Disco Bot to also include the champion's basic info, stats, spells, tips, lore, and skin arts, respectively.

Development

Overview

     is made with Python 2.7 with the Pycharm IDE, as well as various APIs and libraries. I use Discord.py, a python library that wraps Discord's API. For the LoL-specific commands, I use another wrapper library called lolwatcher to access specific data, including player information, and game information. For LoL's static, in-game data, like champions, items, and spells, I directly grab the json files from Riot Games' Static Data API. Disco Bot downloads necessary json files and images from the Static Data API, parses the json and resizes the images, and uploads the data neatly to the text channel.

The output of a champion command. You can see Aurelion Sol's basic info, stats, and spells.

LoL Art

     downloads and resizes the images for the champion's splash art and skin arts using pil and resizeimage. stores the downloaded images in an organized folder. Whenever a command requires an image to be uploaded, checks the folder to see if the image is already downloaded. It also resizes the image if needed.

Aurelion Sol's splash art. This image has been downloaded and cropped.

LoL Game Statistics

    For play statistics, I use a library, py_gg, to access data from champion.gg. This gives access to a variety of statistics, like playrate, winrate, banrate, average kda, average gold, and more.

The top command. has displayed the top 15 support champions, including their winrates, pickrates, and banrates.

Music

    For the music portion, I simply use Discord.py. keeps a linked list queue of songs queued for easy insertions and removals. The bot always plays the song at the front of the queue.

The play or queue command. searches for the most relvant song from the search query "something," and queues up "Something Just Like This." Since it is the first song in the list, begins playing the song.

Future

Current Status

     is currently early in development; thus, it only has basic features, and little to no optimization. At the moment, it can only do these things:

  • Basic music: Queue, play, pause, and skip songs from Youtube and Soundcloud.
  • LoL player statistics: Search up players, and get their winrates, rank, and match history.
  • LoL game data: Search up champions and items, and get their description, info, stats, and art.
  • LoL game statistics: Search up champions and get their in-game statistics, like winrates, playrates, and banrates.
  • Basic math: Add and multiply numbers.
  • Basic dice: Roll die and get their means.

Planned Features

    Currently, the most important thing to do is to refactor. I made while learning how to use Discord's and LoL's APIs; thus, the code is quite messy and unoptimized. Now that I know how the APIs work, I can rewrite the bot from the ground up, organize the functionality into groups, and optimize performance by caching. I am also planning on adding more LoL-specific features, and further into the future, making a text-based RPG played through Discord chat.
